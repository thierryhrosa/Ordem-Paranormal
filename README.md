# Ordo CRIS — Fichas (site estático)

Site simples de fichas inspirado no C.R.I.S — pensado para funcionar no celular e ser hospedado via GitHub Pages.

## Como usar
- Crie/edite fichas no painel.
- Salva automaticamente no **localStorage** do navegador do seu celular.
- Use **Exportar** para baixar JSON e **Importar** para carregar de volta.

## Instalação no GitHub (pelo celular)
1. No seu celular, abra https://github.com e entre na sua conta.
2. Crie um novo repositório (por ex. `ordocris-fichas`) — marque **public**.
3. No repositório recém-criado, clique em **Add file → Create new file**.
4. Crie os arquivos: `index.html`, cole o conteúdo do arquivo `index.html` e commit.
5. Repita para `styles.css`, `app.js` e `README.md`.
6. Após todos os arquivos subidos, vá em **Settings → Pages** (ou **Settings → Code and automation → Pages**).
7. Em **Source**, selecione `main` branch e `/ (root)` e clique **Save**.
8. Aguarde alguns minutos. Seu site estará em: `https://SEU_USUARIO.github.io/NOME_DO_REPO`.

## Dicas (pelo celular)
- Use o app GitHub para criar/editar arquivos se preferir.
- Para editar rapidamente, abra o repositório no Safari/Chrome, clique em **Add file** → **Create new file**.
- Se quiser atualizar o visual: edite `styles.css` e commit.

## Export / Import
- Exporta um JSON por ficha ou todas as fichas.
- Importa um JSON de ficha ou um array de fichas.
